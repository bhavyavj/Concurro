How the whole flow works (what you're watching)

1. Submit
You send a batch of work (URLs or log lines) through the form or CLI.
2. Queue
Items go into a channel queue. The bounded pool of goroutines pulls from it.
3. Concurrent Work
Up to 8 workers run at the same time (real Go goroutines). Watch the viz above light up.
4. Results
Each result is written back instantly. The table below updates live via polling.
